# Copy Building

BepInEx mod for The Planet Crafter that lets you copy an existing building by aiming at it and pressing middle mouse.

## How it works

1. Aim at an existing constructible building.
2. Press middle mouse button.
3. If building is allowed, you enter placement mode for that building.
4. If resources are missing, a cursor message is shown.

The mod uses the game's native `PlayerBuilder.SetNewGhost(...)` path, so mods that alter build checks (such as nearby-container crafting mods) can still affect whether copying is allowed.

## Config

Config file is generated by BepInEx:

- `General.CopyBuildingSettings` (default: `true`)
  - When enabled, copied buildings also attempt to copy these attributes from the source building:
    - `WorldObject.SetSetting(...)` value (used by many machine/selection preferences)
    - linked groups (`SetLinkedGroups`) such as recipe/filter-style selections
    - linked planet (`SetPlanetLinkedHash`) where applicable
    - text (`SetText`) for text-enabled objects
    - color (`SetColor`) for colorable objects
  - The mod applies these directly on `WorldObject` and also through related proxies when present (`SettingProxy`, `TextProxy`, `ColorProxy`, `LinkedPlanetProxy`, `LinkedGroupsProxy`).
- `General.MaxCopyDistance` (default: `60`)
  - Maximum distance for raycast-based target detection when choosing what to copy.

## Build

From this project directory:

```powershell
dotnet build -c Debug
dotnet build -c Release
```

## Deploy

```powershell
Copy-Item ".\bin\Release\netstandard2.1\CopyBuilding.dll" `
  "C:\Program Files (x86)\Steam\steamapps\common\The Planet Crafter\BepInEx\plugins\CopyBuilding.dll" -Force
```
